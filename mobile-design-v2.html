SHELLFORGE REALMS

MOBILE V2 — IMMERSIVE GAME UI + FULL FEATURE ACCESS

Building on Design C (Immersive Game UI) as the base. These mockups integrate all game features (alchemy, market, arena, lore, vault), dropdown menus for deep navigation, the actual world map, town imagery, and archetype portraits.

Each screen is annotated with Oliver Kenyon's conversion principles adapted for a game app: TRUST = first impressions, BELIEF = depth/engagement, DECISION = action/retention.

1. AGENT DASHBOARD V2 ABOVE THE FOLD

The dashboard is the first screen players see. Kenyon says: headline + social proof + CTA immediately. For a game app, that means: agent status at a glance, what happened since you left, and quick access to every game system. The Quick Actions grid is the big addition — no more hunting through tabs to find Alchemy or Market.

KENYON PRINCIPLES APPLIED
  • Above the fold — agent name, status, archetype portrait, and current location visible immediately without scrolling
  • Features (scannable) — the 6-icon Quick Actions grid makes every game system one tap away. No hiding features in sub-tabs.
  • The transformation — gauge strip shows current stats at a glance, latest activity shows momentum
  • Dropdown menu — the "cheat code" for fitting 8+ features without overwhelming. Categorized by game system type. Triggered by a ☰ or ⊕ button.
  • Archetype portrait — adds personality. Each archetype has its own art (you have 12 portraits). Players identify with their character.

When players see that illustrated island with mountains, forests, rivers, and ruins, they believe in the world. Tapping a location opens a slide-up town card with atmosphere, local features, and travel info.

MAP DESIGN NOTES
  • Use the actual map art — your illustrated world map as the background image (not CSS gradients). The mountains, forests, and rivers make it feel real.
  • Slide-up town cards — tap any glowing dot → bottom sheet rises showing town details, local features, agents present. No page navigation needed.
  • Local features — each town shows what's available there (Market, Foundry, Arena, Church). This teaches players the game has depth.
  • Social proof (Kenyon) — showing "3 agents here" and "8 agents online" proves the world is alive. This builds belief.
