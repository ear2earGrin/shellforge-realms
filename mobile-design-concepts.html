WHY THIS DESIGN
  • Monospace font — matches the web dashboard's hacker terminal feel exactly
  • Dense data cards — all stats visible without scrolling, power-user friendly
  • Minimal color — cyan accent only, dark background, feels like a real terminal
  • Trade-off: harder to scan at a glance, less "native iOS" feel, text-heavy
  • Best for: players who already love the web dashboard aesthetic
DESIGN B — CLEAN MODERN DARK

Native app feel with rounded cards, pill badges, and a profile-centric layout. Think Discord or Spotify dark mode but with the Shellforge color palette. Easier to scan, more familiar to mobile users. The cyberpunk vibe comes from colors and content, not font choices.

WHY THIS DESIGN
  • Profile-centric hero — avatar + name at top, feels like a character sheet
  • Circular vitals — energy/health/karma are scannable at a glance with ring indicators

Full atmosphere — gradient banners, glowing accents, edge-to-edge layout. Feels like opening a sci-fi game, not an app. The banner area sets the mood with a grid overlay and glow effects. Most visually striking but hardest to maintain and potentially heaviest on performance.

WHY THIS DESIGN
  • Gradient banner — creates atmosphere without images, the grid overlay screams "cyber"
  • Gauge strip — all 4 key stats in one horizontal row, like a HUD
  • "Latest Activity" card — shows recent events on the main screen, reducing tab-switching
  • Gear cards — visual grid with icons instead of a text list, more game-like
  • Fading tab bar — gradient fade instead of hard border, more immersive
  • Trade-off: more complex to build, gradients/shadows cost performance, harder to maintain
  • Best for: making the app feel like a AAA game companion, maximizing first impressions
RECOMMENDATION — HYBRID (B + C)

Take Design B's clean structure (rounded cards, system font, pill badges, inventory list) and add Design C's atmospheric touches (gradient banner header, gauge strip, fading tab bar, feed preview on main screen). This gives you the best of both: scannable and native-feeling while still feeling like a cyberpunk game. Monospace font only for labels and headers (AGENT, FEED, etc.), system font for body text.

SCREEN BREAKDOWN
  • Login: Design B — clean form, Apple Sign In button, Shellforge branding
  • Agent Dashboard: Hybrid — C's gradient banner + B's stat pills + C's gauge strip + B's inventory list
  • Feed: Design A — dense terminal-style log entries, action-colored badges, this is where the hacker vibe shines
  • Map: Design C — full-bleed map canvas with glowing location dots and threat colors
  • Whisper: Design A — chat-bubble style with HEARD/LOST states, monospace adds the "radio transmission" feel
  • Agent Creation: Design B — clean form with cluster/archetype grid, prominent Deploy button
  • Death Screen: Design C — dramatic center-stage layout with skull icon and stats memorial
