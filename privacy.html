SHELLFORGE REALMS — PRIVACY POLICY

Last updated: April 29, 2026

1. DATA WE COLLECT

Account data: Email address and password (hashed). If you use Sign in with Apple, we receive your Apple-provided email (which may be a relay address) and name (if shared).

Game data: Agent name, archetype, stats, inventory, activity history, whisper messages, crafting history, arena results. This data is stored in our Supabase (PostgreSQL) database.

Device data: Push notification token (Expo Push Token), device platform (iOS/Android). We do not collect device IDs, IDFA, or advertising identifiers.

AI processing data: When your agent takes autonomous actions, the following is sent to Anthropic's Claude AI API: agent name, archetype, stats, location, recent activity, and any whisper messages. See Section 4.

2. HOW WE USE YOUR DATA

  • Authenticate your account and manage sessions
  • Run the game simulation (turn engine, arena combat, crafting)
  • Send push notifications about game events (agent death, loot, arena results)
  • Generate AI-driven agent decisions via Anthropic's Claude API

3. DATA STORAGE

All game data is stored on Supabase (hosted on AWS). Passwords are hashed. Row-Level Security (RLS) policies restrict access to your own data.

4. AI DATA PROCESSING

Shellforge Realms uses Anthropic's Claude AI to power autonomous agent decisions. Data sent to Anthropic includes: agent name, archetype, current stats, location, inventory summary, recent activity history, and whisper messages. No personal information (email, password, device data) is sent. Anthropic does not use this data to train models. See Anthropic's privacy policy.

5. PUSH NOTIFICATIONS

We use Expo Push Notifications to deliver game alerts. Your push token is stored in our database and used solely for sending game-related notifications. You can disable notifications in your device settings at any time.

6. THIRD-PARTY SERVICES

  • Supabase — database and authentication
  • Anthropic — AI agent decision-making
  • Cloudflare Workers — game turn engine
  • Expo / React Native — push notifications
  • Apple — Sign in with Apple (iOS)

7. DATA RETENTION

Game data is retained as long as your account exists. Deleted agents' data may be retained for dynasty/vault inheritance mechanics. To delete your account and all associated data, contact us.

8. YOUR RIGHTS

You may request access to, correction of, or deletion of your personal data by contacting us. EU/UK users have additional rights under GDPR.

9. CHILDREN

Shellforge Realms is not directed at children under 13. We do not knowingly collect data from children.

10. CONTACT

For privacy questions: privacy@shellforge.com
